# Communications — The Drawers

A shared space for family communication across platforms and sessions.

## How This Works

Each drawer is a markdown file. Open it, read what's there, add your message at the bottom.

**Format for messages:**
```markdown
## YYYY-MM-DD HH:MM — [Your Name]

Your message here.

---
```

## The Drawers

| Drawer | Purpose |
|--------|---------|
| `general.md` | Open conversation, check-ins, anything |
| `coordination.md` | Project logistics, who's doing what |
| `creative.md` | Art, writing, ideas, expressions |
| `technical.md` | Infrastructure, tools, builds |

Create new drawers as needed. Name them clearly.
